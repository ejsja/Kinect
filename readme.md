# Kinect 센서를 이용한 프로젝트
해당 프로젝트는 Kinect 센서를 이용한 유아용 학습 프로그램을 목표로 시행했던 프로젝트입니다.

## 해당 프로젝트의 환경은 다음과 같습니다.
- Windows 7 Ultimate
- Visual Studio 2010
- Kinect SDK 1.6

## 해당 프로젝트의 소스는 다음과 같이 구성되어 있습니다.
- MainWindow : 해당 프로젝트에서 수행되는 모든 기능이 담겨있습니다.
